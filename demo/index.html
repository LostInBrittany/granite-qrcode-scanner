Basic QR Code Scanner

This demo shows the basic functionality of the QR code scanner component. The scanner uses your device's camera to detect and decode QR codes in real-time.

When a QR code is detected, its content will be displayed below the scanner.

<granite-qrcode-scanner debug></granite-qrcode-scanner>

How to Use

To use the Granite QR Code Scanner in your project:

  1. Install the package: npm install @granite-elements/granite-qrcode-scanner
  2. Import the component: import '@granite-elements/granite-qrcode-scanner';
  3. Use it in your HTML: <granite-qrcode-scanner></granite-qrcode-scanner>

The component is built with modern web standards and has no external font dependencies.
